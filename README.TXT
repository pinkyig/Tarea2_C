*** Instrucciones de compilacion ***


- El programa debe contar con los siguientes archivos para su correcta ejecucion
    - Manufactura.c y Manufactura.h
    - Mundo.c y Mundo.h
    - Robot.c y Robot.h
    - Io.h
    - FactoryOfNumbers.c


- Se debe ejecutar el comando "make" mediante terminal, esto se debe hacer dentro de la carpeta que contiene todos los archivos de ejecucion
- Luego mediente el comando "ls" se podra observar que se creo un archivo ejecutable con el nombre "FactoryOfNumbers"
- Por ultimo se debera ejecutar el siguiente comando por terminal "./FactoryOfNumbers" para dar paso a la ejecucion del programa.
